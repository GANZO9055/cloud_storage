# Облачное хранилище файлов

Проект выполнен в рамках курса Сергея Жукова: 
https://zhukovsd.github.io/java-backend-learning-course/projects/cloud-file-storage/

Функционал приложения

Работа с пользователями:
- Регистрация
- Авторизация
- Logout

Работа с файлами и папками:
- Загрузка (upload) файлов и папок
- Создание новой пустой папки (аналогично созданию новой папки в проводнике)
- Удаление
- Переименование и перемещение
- Скачивание файлов и папок
